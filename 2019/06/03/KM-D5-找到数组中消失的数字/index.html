找到数组中消失的数字

给定一个范围在 $ 1 ≤ a[i] ≤ n $ ( n = 数组大小 ) 的 整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在 $ [1, n] $ 范围之间没有出现在数组中的数字。

您能在不使用额外空间且时间复杂度为O(n)的情况下完成这个任务吗? 你可以假定返回的数组不算在额外空间内。

有了day2-4的铺垫，这道题就简单多了。简单的说步骤如下：

(1). 判断索引与其对应的元素值是否符合映射关系：$i + 1 = a[i]$ 。

(2). 符合的就跳过，不符合的就一直替换，直到出现两种情况之一：

  1. 换到了和当前索引相对应的元素值；

  2. 与当前元素值对应的索引处的元素值已经满足映射关系，说明当前元素是重复的那个。

(3). 完成替换后再扫描一次数组，其中索引值和对应的元素值构不成映射关系的即消失的数字。
